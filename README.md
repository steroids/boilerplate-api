### Замена названий

Во всем проекте, включая этот файл, замени с учетом регистра на имя проекта:

1. Для человеко-понятного названия: `Boilerplate12345`
2. Для ссылок, названия БД: `boilerplate12345`
3. Измени название проекта в IDE
